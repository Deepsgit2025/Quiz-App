This is a simple quiz application made using **HTML, CSS, and JavaScript**.

The quiz questions come from an online API called **Open Trivia Database**.

## What this project does

- User selects number of questions
- User selects category
- User selects difficulty
- Quiz starts after clicking Start
- User answers multiple-choice questions
- User can go **Next** and **Previous**
- Final score is shown at the end

---

## Technologies used

- HTML – for page structure  
- CSS – for styling the page  
- JavaScript – for quiz logic and API  
- Open Trivia Database API – for questions

---

## Files in this project


index.html → main page
style.css → design and layout
script.js → quiz logic


---

## How to run this project

1. Download or clone the project
2. Open the folder
3. Double click **index.html**
4. Quiz will open in browser

---

## How the quiz works

1. Enter how many questions you want
2. Select category and difficulty
3. Click **Start**
4. Select an answer
5. Use **Next** or **Previous**
6. Click **Submit** to see final score
